Cómo se usa AirPrint

Este documento explica cómo usar AirPrint para imprimir inalámbricamente desde un iPad, iPhone o iPod touch en una impresora Canon.

AirPrint le permite imprimir fotos, mensajes de correo electrónico, páginas Web y documentos desde su dispositivo Apple directamente en la impresora, sin necesidad de instalar ningún controlador.

Entorno de funcionamiento de AirPrint

Para usar AirPrint, es necesario uno de los dispositivos siguientes con la versión de iOS más reciente: iPad (todos los modelos), iPhone (3GS o posterior), iPod touch (3ª generación o posterior).

Entorno de red

El dispositivo Apple (iPad, iPhone o iPod touch) y su impresora deben estar conectados a la misma red Wi-Fi.

Impresión con AirPrint

  1. Compruebe si su impresora está encendida y conectada a la red LAN inalámbrica local.

  2. En el software de la aplicación de su dispositivo Apple, pulse para que aparezcan las opciones de menú.

  3. En el menú desplegable, pulse Imprimir.

  4. En Opciones de impresora, seleccione una impresora.

    5. Importante

    • Hay determinadas aplicaciones de software que pueden no ser compatibles con AirPrint, en estos casos Opciones de impresora podría no aparecer. Si una aplicación de software no le permite usar las opciones de impresora, no podrá imprimir nada desde esa aplicación.

  5. Para 1 copia, haga clic en + o - para definir el número de copias requeridas.

  6. Para la Impresión a doble cara, haga clic en Activar para habilitar la impresión a doble cara, o en Desactivar para deshabilitar la función.

  7. Cuando imprima un tipo de archivo que tiene varias páginas, por ejemplo un archivo PDF, haga clic en Rango y, a continuación, haga clic en Todas las páginas o seleccione el intervalo de páginas que desea imprimir.

    Nota

    • Las opciones de impresora difieren en función de la aplicación de software y la impresora que se estén usando. Normalmente, cuando pulsa Imprimir y selecciona una impresora, las opciones de impresora mostradas incluirán las siguientes cuatro opciones:

      • Impresora: Seleccione una impresora que se pueda usar en su red LAN inalámbrica.

      • Copias: Seleccione el número de copias de impresión.

      • Impresión a doble cara: Si la impresora admite la impresión a doble cara, puede seleccionar Activar o Desactivar.

      • Rango: El intervalo de páginas se puede usar únicamente con aplicaciones de software y tipos de archivo específicos (como los archivos PDF).

  8. Pulse Imprimir.

    Cuando ejecute la impresión, la impresora imprime según la configuración especificada.

Tamaño del papel

Con AirPrint, el tamaño del papel se selecciona automáticamente conforme a la aplicación de software que se está usando en el dispositivo Apple y la región en la que se usa AirPrint.

Cuando se usa el software de la aplicación fotográfica de Apple, el tamaño de papel predeterminado es tamaño L en Japón, y 4"x6" o KG en otros países y regiones.
Cuando los documentos se imprimen desde la aplicación de software Safari de Apple, el tamaño de papel predeterminado es tamaño carta en Estados Unidos, y A4 en Japón y Europa.

Importante

  • Su aplicación de software puede admitir distintos tamaños de papel.

Comprobación del estado de la impresión

Durante la impresión, aparece un icono de impresora en la lista de las aplicaciones de software utilizadas recientemente y puede usarlo para comprobar el avance de la impresión.
Para que aparezca la lista de las aplicaciones de software utilizadas más recientemente, pulse dos veces el botón Inicio en el dispositivo Apple. El icono de impresora aparece en la parte inferior de la ventana Inicio.

Eliminación de un trabajo de impresión

Para eliminar un trabajo de impresión con AirPrint, use uno de los dos métodos siguientes:

  • En el panel de operaciones de la impresora: Use el panel de operaciones para cancelar el trabajo de impresión.

  • En un dispositivo Apple: Pulse dos veces el botón Inicio en el dispositivo Apple. Pulse el icono de impresora y abra Trabajos de impresión para ver la lista de trabajos de impresión. Pulse el trabajo de impresión que desea cancelar y, a continuación, pulse Cancelar impresión.

Solución de problemas de AirPrint

Si el documento no se imprime, compruebe lo siguiente:

  1. Compruebe que la impresora está encendida. Si la impresora está encendida, apáguela y vuelva a encenderla; compruebe si el problema se ha solucionado.

  2. Compruebe si la impresora está conectada mediante la red LAN inalámbrica a la misma subred de red que el dispositivo en el que está instalado iOS.

  3. Compruebe si el firmware de la impresora se ha actualizado a la versión compatible con AirPrint.

  4. Compruebe si la impresora tiene suficiente papel y tinta.

  5. Compruebe que no aparece ningún mensaje de error en el panel de operaciones de la impresora.

    Si el problema no se ha solucionado, use el controlador de impresora de su ordenador para ejecutar la impresión y compruebe si el documento se puede imprimir con normalidad en la impresora.

    Si el documento no se puede imprimir, consulte el manual en pantalla (instrucciones de funcionamiento) que se entrega con la impresora.

Nota

  • Después de encender la impresora, pueden pasar unos minutos antes de que esta pueda establecer la comunicación a través de una conexión LAN inalámbrica. Confirme si la impresora está conectada a la red LAN inalámbrica y vuelva a intentar la impresión.

  • Si Bonjour está deshabilitado, AirPrint no se puede utilizar. Asegúrese de habilitar Bonjour.
